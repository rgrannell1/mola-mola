
# Language Design

## Parser

### Atoms

Ruby-style atoms.

```
:my-atom
```

### Variable Names

```
my-variable <- :my-atom
```

### Booleans

```
#t
#f
#inert
```

### Number

```
-3.14159
```

### Variable Assignments

Mola uses R-style variable assignment

```
dash <- 10
```

### Function Calls

```
max(my-val, 0)
```
